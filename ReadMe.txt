For WINDOWS only( Uses C# and Python to run )

DEPENDENCIES:-
Python 3.5 installed and added to path
requests - in python
python-dateutil

The code for installing the libraries after getting Python3.5
pip install requests
pip install python-dateutil

USE-
1. Make a shortcut of 'MainScript.bat' in the PowerBI_Pipedrive folder
   where ever you find it convenient.
2. Run the shortcut.
3. Type 'u' and enter to update the present dataset.
4. Type 'c' and enter to create a new one.

At present all the bindings are to Skylark_Official_Test dataset.

NOTE- FOR Creating dataset first DELETE the old dataset
